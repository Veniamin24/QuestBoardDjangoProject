## ⚔️ QuestBoard

QuestBoard is a fantasy-themed Django web application for managing heroes, guilds, and quests in a game-inspired world.

## ✨ Features
- CRUD operations for heroes, guilds, and quests
- Hero assignment to quests
- Bootstrap design
- PostgreSQL database

## 🛠 Technologies
- Python
- Django
- PostgreSQL
- Bootstrap 5

## 🚀 Setup
Clone the repository, create and activate a virtual environment, install the required packages, create a PostgreSQL database, update the database settings in `settings.py`, apply migrations, and run the server.
